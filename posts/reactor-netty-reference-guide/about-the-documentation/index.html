这一章节是关于Reactor Netty参考文档的一些简要概述。您可以不用按照顺序阅读整个指南。每一个部分都是独立的，虽然经常会在某一部分引用其他的部分的内容。

1.1.最新的版本和版权声明

Reactor Netty参考指南可以通过HTML文档的形式提供访问。最新的副本可以访问这个网站 https://projectreactor.io/docs/netty/release/reference/index.html

这个文档的副本可以供您自己使用或者提供给其他人使用，但是您不能对该副本收取任何费用并且不管是印刷版还是电子版，每份副本中都必须包含这个版权声明。

1.2.贡献文档

这个参考指南是用Asciidoc写的，您可以通访问 https://github.com/reactor/reactor-netty/tree/master/docs/asciidoc 找到它的原始文件。

如果您有任何的改进，我们将很高兴收到您的pull request。

我们建议您检出版本库到本地，这样您就可以通过asciidoctor Gradle任务和检查渲染来生成这个文档。有一些章节需要依赖于已包含的文件，所以GitHub的渲染并不总是完整的。

为了方便文档的编辑，大部分章节的末尾都有一个链接，可以直接在GitHub上打开该章节的主源文件的编辑界面。这些链接只存在于这个文档的HTML5版本中。他们看起来像这样的：Suggest Edit to About the Documentation 。

1.3. 获得帮助

您可以通过以下几种方式来获取有关Reactor Netty的帮助：

  • 在Gitter 和社区取得联系。
  • 在stackoverflow.com的reactor-netty 板块问问题。
  • 在Github的reactor-netty 中提出错误。

Reactor Netty全部都是开源的，包括这个文档 。
